# Rocket API

A simple dummy API using rocket.

## Install & Run

Install rust using (rustup)[https://rustup.rs/]

```
#Set nightly as default
rustup default nightly

git clone https://github.com/busykoala/rocket_api.git
cd rocket_api
cargo run
```
